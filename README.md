# Virtual Keyboard (Pure JavaScript)

Виртуальная клавиатура, реализованная на чистом JavaScript без использования HTML-разметки для клавиш. Проект построен на классовых ES-модулях с модульной архитектурой.

## Описание проекта

Вся клавиатура динамически создаётся через JavaScript, без заранее прописанной HTML-структуры. Сделано это было для понимания рендеринга компонентов на чистом JS.

## Технологии:
- JavaScript (ES6+)
- Классовые модули
- Без сторонних библиотек и фреймворков
- Без HTML-шаблонов для клавиатуры

## Функциональные возможности

- Ввод текста кликом мыши по виртуальным клавишам  
- Ввод текста с физической клавиатуры  
- Поддержка нескольких раскладок  
- Поддержка разных типов клавиатур  
- Модульная архитектура  
- Логически разделённые слои (рендеринг, логика, обработка событий)


## Возможности для улучшения:
- Добавление возможности кастомизации клавиатуры с сохранением состояния
- Реализовать функцию вставки и копирования текста
- Добавить возможноть выделения текста мышью для быстрого ввода или копирования содержимого


## Навигация по проекту:
Архитектура проекта содержит корневой script.js, который инициализирует приложение, обрабатывает глобальные константные переменные окружения.
Далее содержится папка modules со всеми остальные компонентами клавиатуры, такие как:
- app.js отвечает за обработку событий всей клавиатуры
- keyboard.js сам компонент клавиатуры, который рендерит/ререндерит клавиши
- output.js компонент поля ввода. Обрабатывает всё, что связано с вводом(добавление, удаление, переносы строк и т.д)
- generator.js вспомогательная функция для централизации общей, переиспользуемой логики
- key-config.js содержит настройки служебных клавиш(Tab, delete, space т.д)
- keys.js инициализирует объект, который содержит клавиши и настройки к ним


Каждый модуль отвечает за строго ограниченную область логики, что упрощает чтение кода и его модификацию.

## Запуск проекта

Проект не требует сборки или создания .env. Так же не требуется ставить пакетные менеджеры.

Доступны два способа запуска:

1. Открыть HTML-файл напрямую в браузере
2. Запустить через расширение Live Server
